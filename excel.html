DATA CLEANING IN EXCEL


In this project, I used a US Presidents dataset obtained from Kaggle and performed data cleaning in Microsoft Excel

In this project, I performed data cleaning in Microsoft Excel to prepare a raw dataset for analysis. I began by creating a backup of the original data and removing duplicate records. Next, I standardized the dataset by formatting names with proper capitalization, grouping inconsistent party names, trimming extra spaces, converting currency values into numeric format, and ensuring all dates were stored in a consistent date format. After validating the data for accuracy, I removed unnecessary columns and retained only relevant information, resulting in a clean, well-structured dataset that was ready for analysis and database use.
